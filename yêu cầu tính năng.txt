sử dụng streamlit để hiển thị dữ liệu
api larksuite để call đến các bảng trên
thông tin client_id và client_secret lấy từ file secrets.toml
tham khảo github cũ tương tự của tôi tại @https://github.com/nguyenngothuong/crm-streamlit-luc-thuy
docs search: https://open.larksuite.com/document/uAjLw4CM/ukTMukTMukTM

bước 1 call api larksuite để lấy token
bước 2 call api larksuite để lấy dữ liệu từ các bảng
bước 3 streamlit hiển thị dữ liệu (lọc làm sạch dữ liệu)
